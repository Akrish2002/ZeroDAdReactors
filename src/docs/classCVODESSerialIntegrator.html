Detailed Description

Wrapper for a serial CVODES integration session.

< Prototypes for CVODE fcts., consts.

< Access to serial N_Vector

< Access to dense SUNLinearSolver

< Access to dense SUNMatrix

< Model interface to provide setNEQ(), setInitialState(), evalRHS(), etc.
